# Contributing

Thank you for your interest in TrueHour.

## About This Repository

This repository hosts the **compiled releases** of TrueHour. The application source code is maintained in a private repository.

Contributions here are limited to:

- **Bug reports** — reproducible issues in the released binary
- **Feature requests** — concrete suggestions for upcoming releases

## Reporting Bugs

Use the [Bug Report](https://github.com/itabajah/TrueHour-Releases/issues/new?template=bug_report.yml) template and include:

- App version (from the release tag or Settings screen)
- Windows version and build number
- App language (English / Hebrew / Arabic)
- Numbered steps to reproduce
- Expected vs. actual behavior
- Screenshots or error dialogs when possible

A clear, reproducible report helps us fix issues faster.

## Requesting Features

Use the [Feature Request](https://github.com/itabajah/TrueHour-Releases/issues/new?template=feature_request.yml) template and describe:

- The problem you are solving and why it matters
- Your proposed solution or ideal workflow
- Which area of the app it relates to (import, timesheet, export, etc.)

## Guidelines

- **Search first.** Check [existing issues](https://github.com/itabajah/TrueHour-Releases/issues) before creating a new one.
- **One issue per report.** Keep bug reports and feature requests focused on a single topic.
- **Security issues** should be reported privately — see [SECURITY.md](SECURITY.md).
